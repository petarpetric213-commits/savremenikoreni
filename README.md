# Savremeni Koreni — Online Prodavnica

Luxury e-commerce website for **Savremeni Koreni** (Modern Roots), a brand of handcrafted bags and knitted items. The site combines modern design sensibility with artisanal aesthetics, using a deep forest green and warm gold palette.

## Tech Stack

| Layer | Technology |
|-------|------------|
| Framework | TanStack Start (React 19, SSR) |
| Router | TanStack Router v1 (file-based) |
| Build | Vite 7 |
| Styling | Tailwind CSS 4 + custom CSS variables |
| Fonts | Playfair Display (headings) + Inter (body) |
| Deployment | Netlify |

## Features

- **Multi-page** — Home, Kolekcija (products), Detalj proizvoda, O nama, Kontakt, Korpa, Naručivanje
- **Shopping cart** — localStorage-backed cart with quantity controls
- **Checkout flow** — Order form with contact details, no real payment (orders handled manually)
- **Product filtering** — by category (Torbice / Pleteno)
- **Responsive** — Mobile-first with hamburger navigation
- **Luxury UI** — Playfair Display headings, gold accents, cream backgrounds

## Running Locally

```bash
npm install
npm run dev
```

The site runs on `http://localhost:3000`.
