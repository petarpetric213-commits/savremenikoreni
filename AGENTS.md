# AGENTS.md — Savremeni Koreni

E-commerce site for a handcrafted bags and knitwear brand. Built with TanStack Start on Netlify.

## Tech Stack

| Layer | Technology |
|-------|------------|
| Framework | TanStack Start (SSR + React 19) |
| Router | TanStack Router v1 (file-based routing, auto-generated routeTree) |
| Build | Vite 7 |
| Styling | Tailwind CSS 4 with `@theme` tokens + global CSS utilities |
| Fonts | Playfair Display (headings) + Inter (body) via Google Fonts |
| State | React Context + localStorage for cart |
| Deployment | Netlify |

## Directory Structure

```
public/
  images/          — Product and hero images (PNG)
  favicon.ico
  placeholder.png
src/
  components/
    BuyButton.tsx  — Add-to-cart + go-to-cart buttons
  context/
    CartContext.tsx — Cart state (localStorage persistence, React Context)
  data/
    products.ts    — Product catalog (8 items: bags + knitwear)
  lib/
    stripe.ts      — Stub (Stripe not active; orders handled manually)
  routes/
    __root.tsx     — HTML shell + nav/footer layout + CartProvider
    index.tsx      — Homepage: hero, featured, about teaser, social grid, CTA
    proizvodi.tsx  — Products listing with category filter
    o-nama.tsx     — About page with brand story and values
    kontakt.tsx    — Contact page with form
    korpa.tsx      — Shopping cart page
    products/
      $productId.tsx — Product detail page
    checkout/
      index.tsx    — Checkout form (sends order, no payment)
      success.tsx  — Order confirmation page
      cancel.tsx   — Cancelled order page
  styles.css       — Tailwind import + @theme tokens + utility classes
  router.tsx       — Router setup (reads auto-generated routeTree.gen.ts)
```

## Cart Architecture

- `CartContext.tsx` provides `addItem`, `removeItem`, `updateQuantity`, `clearCart`, `totalItems`, `totalPrice`
- Initial state empty (SSR-safe); loads from `localStorage` on client mount
- Cart icon in header shows live item count
- Checkout form collects contact/address info; on submit clears cart and shows confirmation

## Product Catalog (`src/data/products.ts`)

8 products, category `'torbice'` | `'pleteno'`, prices in RSD. `featured: true` shows on homepage.

## Routing

TanStack Router file-based routing. `routeTree.gen.ts` is auto-generated by the Vite plugin at build time — do not create or edit this file manually.

Routes:
- `/` — Homepage
- `/proizvodi` — All products with filter
- `/products/$productId` — Product detail
- `/o-nama` — About
- `/kontakt` — Contact
- `/korpa` — Cart
- `/checkout` — Checkout form
- `/checkout/success` — Order success
- `/checkout/cancel` — Order cancelled

## Conventions

- TypeScript strict mode — no `any` except explicit casts in form iteration
- `@/` alias maps to `src/`
- All routes use `createFileRoute` with the full path string
- Inline `style={{}}` preferred over Tailwind for brand-specific pixel values
- Tailwind utility classes used for layout (grid, flex, gap, padding)
